## required

* Slack API
  * https://api.slack.com/
* Render
  * https://dashboard.render.com/
* Line Developer - Messaging API
  * https://developers.line.biz/ja/services/messaging-api/
* Node

## start

1. cp .env.sample .env
2. setting .env
3. start
```
yarn install
yarn live
```
